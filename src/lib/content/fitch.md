### BetterFitchVizier
Fitch is a language to write formal logic proofs in. My university had a basic tool to validate these proofs, but the UI was very basic.


I forked it to add more and more features, including syntax highlighting, a shortcut to format, multiple persistent tabs, and much more.

- [Source Code](https://github.com/CodeForGroningen/BetterVizier)
- [Live Version](https://fitch.wireva.eu)
